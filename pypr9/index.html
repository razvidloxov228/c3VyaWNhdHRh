Контрольні питання

  1. FastAPI це сучасний Python‑фреймворк для створення швидких API на Python, який забезпечує автоматичну документацію, валідацію даних і простоту використання.

  2. Запуск додатку здійснюється командою uvicorn main:app --reload

    Особисто в мене так не вийшло, тому я просто дописав до коду import uvicorn

    та if __name__ == "__main__":

    uvicorn.run(app, host="0.0.0.0", port=8000)

  3. Параметр --reload означає, що сервер буде автоматично перезапускатися при зміні коду, що зручно під час розробки.
  4. Параметри шляху є частиною URL і обов’язкові (/items/5), тоді як параметри запиту передаються після ? і необов’язкові (/items/5?q=test).
  5. Автоматична документація доступна за адресами /docs та /redoc, де можна переглядати та тестувати маршрути.
  6. Якщо передати неправильний тип даних у параметр маршруту, FastAPI поверне помилку 422.
